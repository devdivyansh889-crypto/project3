# 🎲 Dice Roll Game (Python Project)

## 📌 Project Overview

This is a simple **menu-driven Dice Roll Game** developed using Python.
The user plays against the computer, where both roll a dice and the result is displayed after each round.

The project is designed using **modular programming** and follows a clean **package architecture**, making the code easy to understand and maintain.

---

## 🎯 Features

* 🎮 Menu-driven system using `while True`
* 🎲 Random dice roll using `random` module
* 🧠 Function-based game logic
* 🗂️ Modular structure (separate files)
* 📊 Displays winner after each round
* ❌ Input validation for better user interaction

---

## 📁 Project Structure

```
game_project/
│
├── main.py        # Menu-driven program (entry point)
├── dice_game.py   # Game logic (function-based)
└── utils.py       # Common variables (optional)
```

## 🚀 How to Run the Project

1. Make sure Python is installed on your system
2. Download or clone this project
3. Open terminal or command prompt
4. Navigate to project folder
5. Run the program:

```
python main.py
```

---

## 🎮 How the Game Works

1. User selects the option from menu
2. Both user and computer roll the dice
3. Dice values are generated randomly (1–6)
4. Result is displayed:

   * Higher number wins
   * Same number → Draw
